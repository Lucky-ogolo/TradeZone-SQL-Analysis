# TradeZone SQL Analysis — HNG Stage 2

## Project Overview
This project was completed as part of the HNG Data Analytics Internship Stage 2 task.
I worked with a PostgreSQL database for TradeZone, a Nigerian e-commerce platform,
cleaning the data and answering 8 business questions to help leadership make informed
decisions for the 2025 planning cycle.

## Dataset
- 7 tables: customers, sellers, products, orders, order_items, payments, reviews
- 865 customers, 3,015 orders, 280 products across 5 Nigerian states
- Data covers 2023 to 2024

## Tools Used
- PostgreSQL (pgAdmin 4)

## What I Did
- Cleaned and standardised the database (Part 1.sql)
- Answered 8 business questions using SQL queries (Q1 to Q8)
- Delivered a professional analyst memo with findings and recommendations

## Key Findings
- Lagos had the highest customer acquisition with a 49% conversion rate
- All top 10 revenue generating products were from the Electronics category
- Payment preferences varied significantly across states

## Files in this Repository
- Data cleaning script
-  Business question queries
- Cleaned database export
